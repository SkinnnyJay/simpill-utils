# Examples

Runnable examples for `@simpill/file.utils` (Node.js server).

## Quick Start

```bash
# From the file.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | readFileUtf8, writeFileUtf8, readFileJson, writeFileJson, ensureDir |

See the [main README](../README.md) for full API documentation.
