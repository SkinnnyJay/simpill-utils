# Examples

Runnable examples for `@simpill/object.utils`.

## Quick Start

```bash
# From the object.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | pick, omit, getByPath, deepMerge, isPlainObject |
