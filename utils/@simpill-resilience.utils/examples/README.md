# Examples

Runnable examples for `@simpill/resilience.utils`.

## Quick Start

```bash
# From the resilience.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | CircuitBreaker, RateLimiter, createBulkhead, withJitter |
