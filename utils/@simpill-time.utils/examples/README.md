# Examples

Runnable examples for `@simpill/time.utils`.

## Quick Start

```bash
# From the time.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | getUnixTimeStamp, addDays, diff, debounce, throttle |

See the [main README](../README.md) for full API documentation.
