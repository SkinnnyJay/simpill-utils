# Examples

Runnable examples for `@simpill/patterns.utils`.

## Quick Start

```bash
# From the patterns.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
npx ts-node examples/02-core-patterns.ts
npx ts-node examples/03-structure-proxy.ts
npx ts-node examples/04-creational.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | Result/Either, strategy selector, async pipeline |
| [02-core-patterns.ts](./02-core-patterns.ts) | Adapter, Decorator, Observer, State machine, Mediator |
| [03-structure-proxy.ts](./03-structure-proxy.ts) | Composite, Proxy |
| [04-creational.ts](./04-creational.ts) | Builder, Facade, Flyweight |

See the [main README](../README.md) for full API documentation.
