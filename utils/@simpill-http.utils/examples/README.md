# Examples

Runnable examples for `@simpill/http.utils`.

## Quick Start

```bash
# From the http.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | fetchWithTimeout, fetchWithRetry, createHttpClient, isRetryableStatus |
