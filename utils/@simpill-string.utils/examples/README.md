# Examples

Runnable examples for `@simpill/string.utils`.

## Quick Start

```bash
# From the string.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | StringBuilder, formatString, toCamelCase, truncate, slugify |

See the [main README](../README.md) for full API documentation.
