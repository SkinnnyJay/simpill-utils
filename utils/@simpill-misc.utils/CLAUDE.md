# misc.utils

Convenience bundle: re-exports only. No local implementations. Canonical packages: function.utils (debounce, throttle, once), async.utils (raceWithTimeout, PollingManager), cache.utils (memoize), object.utils (singleton, BoundedArray, BoundedLRUMap), uuid.utils, enum.utils, time.utils (IntervalManager, timers).

- Main: `src/index.ts`; client: `src/client/index.ts`; server: `src/server/index.ts`; shared: `src/shared/index.ts`.
- Commands: `npm run build`, `npm test`, `npm run verify`.
