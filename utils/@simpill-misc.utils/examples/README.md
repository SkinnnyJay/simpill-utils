# Examples

Runnable examples for `@simpill/misc.utils`.

## Quick Start

```bash
# From the misc.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | singleton, debounce, throttle, once, memoize, LRU |
