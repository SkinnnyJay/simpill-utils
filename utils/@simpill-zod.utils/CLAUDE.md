# @simpill/zod.utils

Zod helpers package: schema builders, safe-parse result helpers, common transforms, OpenAPI metadata.

**Commands**: `npm run build`, `npm test`, `npm run test:coverage`, `npm run check:fix`, `npm run verify`.

**Exports**: Main barrel and subpaths `./shared`, `./client`, `./server`. Implementations live in `src/shared/`; client/server only for runtime-specific code.

**Testing**: Jest, 80% coverage. Tests in `__tests__/shared/unit/` and `__tests__/shared/integration/`.
