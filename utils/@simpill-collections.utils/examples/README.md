# Examples

Runnable examples for `@simpill/collections.utils`.

## Quick Start

```bash
# From the collections.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | Queue, Stack, LinkedList, Vector, Deque, CircularBuffer, LRUCache, TTLCache, MultiMap, BiMap, OrderedMap, TypedSet |

See the [main README](../README.md) for full API documentation.
