# Benchmark data

Generated by `npm run benchmark:data` from `utils/token-optimizer.utils`.

## Layout

- `{format}-{size}mb.{ext}` – Sample files (CSV, TXT, JSON, MD, XML, JSONL) at 1MB, 5MB, 20MB.
- `manifest.json` – Paths, formats, target/actual byte sizes, and seed used for generation.
- `reports/` – Benchmark JSON reports (written by `npm run benchmark`).

## Generation

Uses `@faker-js/faker` with a fixed seed (default `42`, override with `BENCHMARK_SEED`) so runs are reproducible. Sizes are approximate; actual bytes are recorded in the manifest.
