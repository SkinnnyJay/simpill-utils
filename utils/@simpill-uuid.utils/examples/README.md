# Examples

Runnable examples for `@simpill/uuid.utils`.

## Quick Start

```bash
# From the uuid.utils directory
npm install
npx ts-node examples/01-basic-usage.ts
```

## Examples

| Example | Description |
|---------|-------------|
| [01-basic-usage.ts](./01-basic-usage.ts) | generateUUID, validateUUID, isUUID, compareUUIDs |

See the [main README](../README.md) for full API documentation.
